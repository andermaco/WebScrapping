Web Scraping sample for Android based on Material Design guide lines. The scrapping results will be shown inside swipe views with tabs: links and imports.

Sample Espresso functional test has been added at: WebScrapping/app/src/androidTest/java/com/andermaco/scrapping/MainActivityLayoutTest.java

In order to test the app, please assure the url you are trying to scrap has been set at: WebScrapping/app/src/main/res/values/strings.xml at "url" attribute.
